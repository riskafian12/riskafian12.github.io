Implementasi kemanan jaringan pada mail server zimbra di ubuntu

Pada project ini saya berkelompok dengan 2 orang, mengimplementasikan keamanan jaringan email dengan spamassasin dan blacklist pada localhost server zimbra di OS Ubuntu 18. Zimbra server adalah server open source untuk mengelola email kita dapat membuat domain sendiri seperti pada project ini meggunakan @kamjartubes.net. Kemudian pada sisi keamanannya menggunakan fitur zimbra yaitu spammassasin dan blacklist. Spamassasin memungkinkan kita dapat memblokir kata-kata yang ingin kita blokir pada pesan yang terkirim ke email kita sehingga pesan tersebut tidak ada pada inbox mail. Blacklist memungkinkan kita memblokir alamat email seseorang sehingga tidak dapat mengirim email ke kita.

Dalam project ini saya bertugas untuk mengkonfigurasi dan menginstall server zimbra

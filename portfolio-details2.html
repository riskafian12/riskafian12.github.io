Pada project ini saya kerja praktek selama satu bulan di Tel-U Coffee. Tugas saya dibagian Front-End. mengupdate informasi, mengganti struktur web, mengganti visual website, dll. Pada project ini saya menggunakan HTML untuk mengupdate informasi web dan mengganti strukttur web, CSS untuk mengganti visual website dan bantuan tools photoshop.

Pada project ini website dari dea.telkomuniversity.ac.id menggunakan library bootstrap untuk tampilan visualnya, untuk strukturnya menggunakan bahasa HTML dan PHP. Tools photoshop sendiri saya gunakan untuk mengedit logo dan icon pada website tersebut. Sistem databasenya sendiri menggunakan MYSQL untuk menyimpan data user seperti username, email, kontak pribadi, dll.

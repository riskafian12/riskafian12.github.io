Automatic Gate and Portal Using RFID for Apartment

Pada project team zeus dengan anggota 5 orang, membuat gerbang dan portal automatis berbasis microcontroller dan web sebagai monitoring dan controlling. Saya bertugas untuk membuat sistem microcontrollernya, Terdapat dua sistem microcontroller yaitu gate dan portal.

Pada sisi gate akan terbuka automatis setiap jam 7 pagi dan tertutup jam 10 malam, dengan modul RTC (modul untuk menyimpan waktu secara realtime) yang dihubungkan dengan NodeMcu dan 2 servo. Kemudian arduino diprogram untuk membaca data waktu pada modul RTC dan diprogram dengan bahasa C/C++ untuk menggerakan servo, ketika modul RTC menunjukan pukul 7.00 NodeMCU akan membaca data tersebut kemudian akan menggerakan servo dengan 90 derajat membuat gerbang membuka dengan bantuan pushroot yang dipasangkan pada servo dan gerbang. Begitu pun sebaliknya ketika waktu menunjukan pukul 22.00 servo akan bergerak 0 derajat sehingga menutup gerbang. Kemudian pada sisi web untuk menghindari sistem failure admin dapat mengcontroll buka tutup gerbang menggunakan web, dengan menggunakan firebase (database yang disinkronkan secara realtime) dihubungkan NodeMcu dengan Auth Key dari firebase dan meenginput data nama hotspot dan password sehingga NodeMcu terhubung internet.

Pada sisi portal saya membuat portal dapat terbuka dengan sistem RFID yang sudah didaftarkan sebelumnya pada web, dengan menggunakan modul 2 MRFC (modul untuk membaca RFID) yang dihubungkan dengan Arduino dan 2 servo. Ketika pengunjung masuk dengan menempelkan kartu RFID yang sudah didaftarkan sebelumnya pada web. Arduino akan membaca dan mengirim data pada XAMPP dengan bahasa python, kemudian ketika data RFID ditemukan pada XAMPP Arduino akan menggerakan servo 90 derajat sehingga portal terbuka dan data akan terekam pada web siapa saja yang masuk lewat portal tersebut. Selain itu pada sisi web dapat mendaftarkan kartu RFID-nya dengan menambahkan pada database web.
